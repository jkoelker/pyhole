pyhole - A modular IRC bot for Python developers.


Core Requirements

- Python 2.5+
- BeautifulSoup
- Eventlet


Plugin Requirements

- launchpadlib
- pywunderground


Test Requirements

- nose

Installation
````
git clone git://github.com/jk0/pyhole.git
cd pyhole
sudo pip install -r requirements.txt
python setup.py install
pyhole
````
